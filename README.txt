Gloom - port Boom3D Quake like engine for BeOS

This is my port Boom3D engine for BeOS. I use DirectWindow for quick draw window.

Keys:

Cursor keys - move to room,
SPACE, D - jump,
C - down,
TAB + Cursor keys - quick move,
A - look up,
Z - look down,
< - move left,
> - move right,
[ - rotate screen left 
] - rotate screen right
